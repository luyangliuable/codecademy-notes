Type of data is the structural and mathematical interpretation. 

* Data Level
  * An item
  * A link
  * An attribute
  
* Dataset level
  * Data type combined together into a larger structure
  * a table
  * a tree
  * a field
  * sampled values

## Semantics of Data
The real life meaning of data.

E.g. Does a word represent someone's first name or fruit?
